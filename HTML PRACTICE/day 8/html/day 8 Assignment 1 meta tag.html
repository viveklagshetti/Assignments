Meta Tags syntax and description

1. Charset Meta tag

Syntax

<meta charset="UTF-8">

Description

This tag specifies the character encoding for the HTML document. UTF-8 is the recommended encoding, as it supports a wide range of characters from various languages.


2. Viewport Meta Tag

Syntax

<meta name="viewport" content="width=device-width, initial-scale=1.0" >

Description

This tag is used to control the viewport's dimensions on different devices, helping to create responsive and mobile-friendly designs


3. Title Meta Tag

Syntax

<meta name="title" content="Page Title" >

Description

This tag sets the title of the webpage that appears in the browser's title bar or tab.


4. Description Meta Tag

Syntax

<meta name="description" content="Description of the page" >

Description

This tag provides a brief description of the content of the page. Search engines often display this description in search results


5. Keywords Meta Tag

Syntax

<meta name="Keywords" content="Keywords1, Keywords2, Keywords3" >

Description

This tag used to be important for SEO, but its impact has diminished. It used to contain a list of keywords related to the page's content


6. Author Meta Tag

Syntax

<meta name="Author" content="Author's name" >

Description

This tag is specifies the Author of that webpage's content
